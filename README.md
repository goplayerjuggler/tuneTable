# Tune table
An HTML viewer for lists of tunes. It uses  data in a hybrid format where the tune can be given in ABC format or more simply just with some metadata. 

Features: 
* a popup score viewer with transpose buttons, for tunes where the abc is included
* sorting and filtering of the list of tunes
* can add links to audio references to the tunes along with notes

## Setup
```
npm install
npm run dev
npm run build
```
